# About
A countdown timer app that allows speed changes of 1x, 1.5x and 2x.

#### Technologies used
- 1 React
- 2 AWS amplify

#### Setup
- 1 clone this repo.
- 2 cd `countdown-timer` and run `npm install`
- 3 run `npm start`
- 4 view app at http://localhost:3000
